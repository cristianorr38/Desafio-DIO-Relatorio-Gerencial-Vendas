## 📌 Instruções de Entrega do Desafio
- **Descrição do desafio:** Você irá criar um relatório mais elaborado com base na *sample financials* do Power BI.

### 📂 Arquivos de Dados
Os arquivos de dados estão disponíveis no GitHub:  
- [Repositório Power BI Analyst](https://github.com/julianazanelatto/power_bi_analyst)

## 🛠️ Passo a Passo:

- 🖼️ Objetos que definem o layout do relatório  
- 📈 Gráficos (*visuais*) e os campos que os compõem  
- 🔘 Botões para navegabilidade

## ✅ Lembrete:
- ➕ Criar a segunda página do relatório  
- ☁️ Publicar o seu relatório no Power BI Service  
- 🎨 Caso tenha familiaridade, fique livre para utilizar outros artifícios nos botões e demais elementos  
- 🔗 Submeter seu projeto através do link no GitHub  

---

## ⚙️ Requisitos
Para executar e visualizar este projeto corretamente, você precisará de:

- 🖥️ **Power BI Desktop** (versão recomendada: 2.120 ou superior)  
- 📂 **Arquivos de dados**:  
  - `Financial Sample.xlsx`  
  - `Relatorio Gerencial Vendas.pbix`  
- 🌐 **Conta no Power BI Service** para publicação e compartilhamento online  
- 🔗 **GitHub** para versionamento e hospedagem do repositório  
- 📱 (Opcional) Aplicativo **Power BI Mobile** para visualização em dispositivos móveis
